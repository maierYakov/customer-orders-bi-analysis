# Customer Orders BI Analysis

BI-проект по анализу клиентов, заказов, оплат и ключевых бизнес-метрик с помощью SQL.

## Цель проекта

Проанализировать данные по клиентам и заказам, рассчитать основные бизнес-показатели и подготовить данные для BI-отчётности.

## Задачи проекта

- Изучить структуру данных.
- Объединить данные клиентов и заказов.
- Рассчитать основные бизнес-метрики.
- Проанализировать выручку по городам.
- Определить активных клиентов.
- Найти клиентов без заказов.
- Сегментировать заказы по размеру оплаты.
- Подготовить аналитические таблицы для отчётности.
- Сформулировать выводы по результатам анализа.

## Основные метрики

- Количество заказов
- Количество оплаченных заказов
- Общая сумма оплат
- Средний чек
- Доля оплаченных заказов
- Количество клиентов
- Количество активных клиентов
- Выручка по городам
- Сегменты заказов

## Что было сделано

В рамках проекта были подготовлены SQL-запросы для анализа клиентской активности, заказов и оплат.

Были рассчитаны ключевые показатели: количество заказов, количество оплаченных заказов, сумма оплат, средний чек и доля оплаченных заказов.

Также была выполнена сегментация заказов по размеру оплаты и подготовлена итоговая таблица, которую можно использовать как основу для BI-отчёта или дашборда.

## Практическая польза

Проект показывает умение:

- работать со связанными таблицами;
- объединять данные с помощью JOIN;
- рассчитывать бизнес-метрики;
- использовать группировку и агрегатные функции;
- применять CASE для сегментации данных;
- готовить данные для отчётности и визуализации.
