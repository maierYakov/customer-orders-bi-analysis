# Выводы по проекту

В рамках проекта был проведён анализ клиентской активности, заказов и оплат.

## Что было сделано

- Изучена структура данных по клиентам и заказам.
- Подготовлены SQL-запросы для объединения таблиц `customers` и `orders`.
- Рассчитаны основные бизнес-метрики: количество заказов, количество оплаченных заказов, общая сумма оплат и средний чек.
- Выполнен анализ выручки по городам.
- Проведена сегментация заказов по размеру оплаты.
- Выделены активные клиенты и клиенты без заказов.
- Подготовлена итоговая таблица, которую можно использовать как основу для BI-отчёта.

## Основные выводы

- Анализ количества заказов помогает оценить общую активность клиентов.
- Расчёт суммы оплат и среднего чека позволяет оценить финансовую эффективность заказов.
- Анализ по городам помогает определить направления с большей выручкой и клиентской активностью.
- Сегментация заказов по размеру оплаты показывает, какие группы заказов дают больший вклад в выручку.
- Поиск клиентов без заказов может быть полезен для задач повторной коммуникации и повышения активности клиентской базы.
- Итоговые SQL-запросы можно использовать для подготовки данных к визуализации в BI-системе.

## Практическая польза проекта

Проект может быть использован как основа для построения BI-дашборда, в котором можно отобразить:

- общую выручку;
- количество заказов;
- количество оплаченных заказов;
- средний чек;
- долю оплаченных заказов;
- выручку по городам;
- активных клиентов;
- сегменты заказов.

## Итог

Проект показывает навыки работы с SQL, понимание базовых бизнес-метрик, умение объединять данные из нескольких таблиц, рассчитывать показатели и подготавливать данные для отчётности.
