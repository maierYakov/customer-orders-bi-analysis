# Описание метрик

В проекте используются основные бизнес-метрики для анализа заказов, клиентов и оплат.

## Orders Count

Количество всех заказов.

Метрика показывает общий объём заказов и помогает оценить активность клиентов.

## Paid Orders Count

Количество заказов со статусом `paid`.

Метрика показывает, сколько заказов были успешно оплачены.

## Total Revenue

Общая сумма оплаченных заказов.

Формула:

SUM(amount), где status = 'paid'

Метрика используется для оценки общей выручки.

## Average Order Value

Средний чек.

Формула:

total_revenue / paid_orders_count

Метрика показывает среднюю сумму оплаченного заказа.

## Payment Conversion Rate

Доля оплаченных заказов среди всех заказов.

Формула:

paid_orders_count / orders_count * 100

Метрика показывает, какая часть заказов завершилась оплатой.

## Customers Count

Количество клиентов.

Метрика помогает оценить размер клиентской базы.

## Active Customers

Количество клиентов, у которых есть хотя бы один заказ.

Метрика показывает, какая часть клиентов проявляет активность.

## Revenue by City

Выручка по городам.

Метрика помогает определить города с наибольшей суммой оплат и клиентской активностью.

## Order Segment

Сегмент заказа по размеру оплаты.

В проекте используются три сегмента:

| Сегмент | Условие | Значение |
|---|---|---|
| high | amount >= 1000 | крупный заказ |
| medium | amount >= 500 | средний заказ |
| low | amount < 500 | небольшой заказ |

Сегментация помогает сравнить вклад разных групп заказов в общую выручку.
